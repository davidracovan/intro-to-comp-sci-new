Basic HTML tags

  • doctype: The document tag specifies what version of HTML we are using so that the browser can render the document effectively.
  • html: This tag defines the root for the document and acts as the container for all other HTML tags.
  • head: This tag defines the metadata for the document.
  • meta charset: This tag defines the type of characters the document is using.
  • body: This tag is the wrapper for all other tags that are visible on the website.
